# Wastage-dashboard
This project analyzes retail product wastage using Microsoft Excel to identify key loss drivers across banners, product categories, and stores.
The goal is to support data-driven decision making by highlighting where wastage is concentrated and which business units require intervention.
The dashboard is designed for commercial managers, supply chain teams, and suppliers to quickly assess wastage performance.
